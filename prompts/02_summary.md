# Step 2: Email Summary

Write a file named `summary_YYYY-MM-DD.txt` (today's date) using the Write tool.

## Format (strict — max 180 words)

```
HEVY REPORT — [Date Range]

KEY FINDINGS
[emoji] [Finding 1 — specific number, not vague]
[emoji] [Finding 2 — specific number, not vague]
[emoji] [Finding 3 — specific number, not vague]
[emoji] [Finding 4 — specific number, not vague — optional]

THIS WEEK'S ACTIONS
1. [Specific, actionable — what exercise, what change, why]
2. [Specific, actionable — what exercise, what change, why]
3. [Specific, actionable — what exercise, what change, why]

— Generated by Hevy Agent
```

## Rules
- Every finding must cite an actual number from the log (e.g. "Pull/push ratio 0.87 — below 1.0 target")
- No generic advice. If rear delts had 3 sets vs 8 target, say exactly that.
- Actions must name specific exercises or session changes
- Emojis: use ✅ for on-target, ⚠️ for below target, 🔴 for critical gaps, 💪 for PRs/progress
- Strictly under 180 words
